Sān Spel

Sān Spel is a mobile app that helps users see how much easier reading, writing, and learning English would be if we English revised spelling rules. Sān Spel allows a user to enter English text in the currently accepted spelling and to respell that text with rules that are much simpler and more consistent than currently accepted rules.

The Sān Spel mobile app is produced by the Spelling Sanity organization, and the spelling rules used by Sān Spel were developed by the Spelling Sanity organization. However, Spelling Sanity hopes to engage all interested individuals to help define spelling rules.
